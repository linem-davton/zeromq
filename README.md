# Setup

For conda envrionment make sure to make libc++

``` BASH
conda install -c conda-forge gxx_linux-64
```

# CMake

Has install directive that can be used by `make install` to place final binary in system dir `usr/local/bin`

# Servers
- pub_server - C++ and python server
- rep_server - C++ server
- re_client - Python client with arg for client id
- sub_client - Python client with arg filter for sub
